Examples
--------

.. code-block:: yaml+jinja

    
    # Create a new LaunchDarkly Project with tags
    - launchdarkly_project:
        state: present
        project_key: test-project-1
        color: C9C9C9
        tags:
          - dev
          - ops
          - frontend
