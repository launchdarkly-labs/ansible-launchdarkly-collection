Examples
--------

.. code-block:: yaml+jinja

    
    # Create a new LaunchDarkly Project with tags
      - name: Generate Test JSON
        launchdarkly_test_generator:
          sdk_key: sdk-test-123456
          overrides_flag:
            - example_test_flag: True
        register: results
