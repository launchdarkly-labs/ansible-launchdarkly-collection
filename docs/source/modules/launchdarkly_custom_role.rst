Examples
--------

.. code-block:: yaml+jinja

    
    - launchdarkly_custom_role:
        state: present
        name: "My Custom Role"
        policies:
            resources:
              - "proj/*:env/*:flag/test_flag"
            actions:
              - "*"
            effect: allow
